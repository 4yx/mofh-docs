Forcing Web Traffic To Use HTTPS

Have you finished installing your SSL certificate on your domain by following this guide? Congratulations upon securing your website!

https://docs.pleasantfree.com/ssl-and-security/install-ssl-certificate

Now you may want to force/forward all of your visitors to use the HTTPS version of your site instead of HTTP. This guide covers all possible methods to do that safely on free hosting.

Forcing HTTPS Using .htaccess​

Get into your File Manager via the client area or control panel. Hover into the /htdocs folder and create a file named .htaccess if it’s not there already, if it is, just edit the file and add the following code into it:

RewriteEngine On 
RewriteCond %{HTTPS} off 
RewriteRule ^(.*)$ https://%{HTTP_HOST}%{REQUEST_URI} [L,R=301]

Ah, don’t forget to save the file!

Edit Your Cloudflare Settings​

If you’re using Cloudflare on your website, you can let Cloudflare enforce HTTPS for you. Simply log in to Cloudflare’s website, go to the SSL/TLS settings for your domain, and enable “Always use HTTPS”.

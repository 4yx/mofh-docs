Common MySQL Errors

MariaDB is the first choice of PHP developers. As an open-source Relational Database Management System (RDBMS) that uses SQL language, MariaDB database helps to automate data retrieving and provides great support in PHP MySQL web application development. As no human is perfext, mistakes can be made along the way. This article will help you to understand the common MySQL errors and how to fix them.

Access denied​

This error message can look like this:

Access denied for user 'cwrt_12345678'@'192.168.%' (using password: YES)

This means that you don't have access to the database. Usually, this can be fixed by ensuring that all credentials are correct and that the databse exists.

Lost connection to MySQL server​

Network conditions should be checked if this is a frequent error. If an error message like “Lost connection to MySQL server” appears while querying the database, it is certain that the error has occurred because of network connection issues. If you do see this error, please report it on the community forum.

The table is full​

This means that you have hit either the inode, disk, or database limit. Because, after all, the database is just a file which takes up space in your Hosting Account.

To "fix" this, try following some steps in our "Disk Quota Exceeded" article.

Table tbl_name doesn’t exist​

In some cases, you may be referring to the table incorrectly. This is possible because the MySQL server uses directories and files for storing database tables. Depending upon the operating system file management, the database and table names can be case sensitive.

Fixing this issue is quite easy: ensure the table exists in the database and that the table name is correct in your code.

Tips to Avoid Errors

Be sure you are using your hosting account password​

The most common issue is that you are not using the right password. You should use the hosting account password in your script configuration.

The hosting account password is not your Client Area password! The hosting account password is randomly generated when you create an account, and you can find or change it in your Client Area (click the Manage button in the card for the account).

You weren’t able to fix the error?​

This is why the Community Forum is here! Open a new topic and don’t forget to explain your issue clearly so that you can get help from the community quickly and continue your mission.
